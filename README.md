Description
 Image Steganography project that allows you to hide secret messages inside images and retrieve them later without noticeable changes to the image.

 📌 What is Steganography?

Steganography is a technique used to conceal information within another file (like an image), making the message invisible to others.

Unlike encryption, the goal is to hide the existence of the message itself.

🚀 Features
🔏 Encode (hide) secret messages inside images
🔓 Decode (extract) hidden messages
🖼️ Works with PNG images (recommended for lossless quality)
⚡ Fast and simple implementation
🧠 Uses LSB (Least Significant Bit) technique
